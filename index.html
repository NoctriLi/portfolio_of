This tool was designed to provide players with actionable insights by analyzing their gameplay data, this data was then filtered to specifications provided by the user. The bot creates a private channel between the user and the admins, which acts as a cart and a chatroom.

The bot will save the receipts in a database, send a copy to the user, and if connected directly write the receipt info into a google spreadsheet.
