"Store Bot"

A shop system bot for Discord designed to facilitate the selling and buying of virtual items among community members.

It incorporates a clever menu system to allow shop administration to manage data features, such as:

  • Adding/removing/editing items to the shop
  • Changing the shop title/description
  • Changing channel the shop is in
  • Connecting a google sheet for logging

The bot is written in javascript and utilizes a mysql server to manage each server's settings and receipts.

Tech stack:

  • Node.js
  • Discord.js
  • MySQL
  • Google Sheets API
